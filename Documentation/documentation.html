HTML Structure

In this template i used as a framework, the Bootstrap v3.0.3 Framework. You can take a look here http://www.getbootstrap.com/ and see it's awesome feature in action

I mostly kept the generic styles, resets & grid. 

You can also see a working example of how the grid system works http://getbootstrap.com/getting-started/#examples-framework

So as a short description, the grid is composed by 12 columns, named like "col-xs-1", "col-xs-2", "col-xs-3"...."col-xs-12". The div's should be placed in "content" div.

A basic layout of the grids look like:

  

<section class="your-class-for-div-container">
    <div class="content">
        <div class="col-xs-5"> 5 Columns </div>
        <div class="col-xs-3"> 3 Columns </div>
        <div class="col-xs-4"> 4 Columns </div>
        <div class="col-xs-12">
          <div class="content">
            <div class="col-xs-6"> 6 columns  </div>
            <div class="col-xs-6"> 6 columns  </div>
          </div>
        </div>
    </div>
</section>
 
Also, remember that there's no need  to add "alpha" & "omega" classes to every row's first & last columns. The framworks knows on it's own when to apply.

CSS Files and Structure

I'm using several CSS files in this theme:


- Generic styles : "css/reset.css"   "css/style.css"   "css/bootstrap.css"   "css/font-awesome.css"  

"css/animate.min.css"  - this stylesheet Is CSS3 animations Plugin, to see examples for this plugin click here : http://daneden.github.io/animate.css/.

"css/owl.carousel.css"  - this is the css style for jQuery OwlCarousel Plugin, it contains all the styles for this plugin. to see examples and how to use this plugin click here : http://owlgraphic.com/owlcarousel/

"css/prettyPhoto.css"  - this is the css style for jQuery prettyPhoto Plugin, is a jQuery lightbox clone. Not only does it support images, it also support for videos, flash, YouTube, iframes and ajax. It’s a full blown media lightbox. to see examples and how to use this plugin click here : http://www.no-margin-for-errors.com/projects/prettyphoto-jquery-lightbox-clone/

"css/settings.css"  - this is the css style for Revolution Responsive Slider jQuery Plugin, : http://codecanyon.net/item/slider-revolution-responsive-jquery-plugin/2580848

"css/responsive.css"  - this stylesheet contains responsive rules for adapting the template to any resolution.

CHANGING FONTS

To change the embeded font, please take a look in the head part of the main css "css/style.css"  and you should find this line:




	
/* Fonts */
	@import url(http://fonts.googleapis.com/css?family=Roboto+Condensed:400,300,700);
    








	To change the font, first go to http://www.google.com/fonts choose a font and use the generated code.



	To add/remove the elements that use the custom font, open "css/style.css" go to line 18 and line 27 you should find the css lines which defines the elements that use the font.

PARALLAX EFFECT


The Parallax Design is a scrolling technique which helps in conjuring the illusion of the depth of field to the user. The technique uses a combination of multiple layers of information presented through graphics and texts. These clusters of information are sped up to create the effect of movement, depth and 3 dimensional illusions. With one flip of the scrolling wheel, the user can get hold of this illusion. This illusion can be used to immerse the website visitor into absolute interactivity of the website.
